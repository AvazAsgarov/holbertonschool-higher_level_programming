# Python - Inheritance

This module contains tasks related to Python inheritance and object-oriented programming concepts.

## Tasks Overview

- **0-lookup.py**: Function to return the list of available attributes and methods of an object
- **1-my_list.py**: MyList class that inherits from list with a print_sorted method
- **2-is_same_class.py**: Function to check if an object is exactly an instance of a specified class
- **3-is_kind_of_class.py**: Function to check if an object is an instance of a class or its subclasses
- **4-inherits_from.py**: Function to check if an object is an instance of an inherited class
- **5-base_geometry.py**: BaseGeometry class with area method
- **6-base_geometry.py**: BaseGeometry class with integer_validator method
- **7-base_geometry.py**: BaseGeometry class with both area and integer_validator methods
- **8-rectangle.py**: Rectangle class that inherits from BaseGeometry
- **9-rectangle.py**: Rectangle class with additional methods
- **10-square.py**: Square class that inherits from Rectangle
- **11-square.py**: Square class with additional features

## Testing
